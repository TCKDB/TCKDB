authors_species¶

An attribute that establishes a bidirectional relationship in a Many to Many data model with the Species table representing species authored by persons in this object. This attribute is added automatically and is only defined under Species as authors.

Type

relationship

reviewers_species¶

An attribute that establishes a bidirectional relationship in a Many to Many data model with the Species table representing species reviewed by persons in this object. This attribute is added automatically and is only defined under Species as reviewers.
