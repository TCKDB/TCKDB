class tckdb.backend.app.models.ess.ESS(**kwargs)[source]¶

A class for representing a TCKDB ESS item

(ESS = electronic structure software)

Examples:

ESS(name='Psi4',
    version='1.1',
    url='http://www.psicode.org/')

ESS(name='Gaussian',
    version='16',
    revision='C.01',
    url='https://gaussian.com/')
